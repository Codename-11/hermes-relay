# 功能概览

## Android

- **流式聊天**：实时 Markdown、推理显示、工具进度、会话历史、附件和发送队列。
- **管理 Hermes**：模型、密钥、技能、配置文件和 `SOUL.md`。
- **语音**：标准 Hermes 语音；配对 Relay 后可使用额外提供商和实时代理模式。
- **多连接**：在家庭、工作、开发和生产 Hermes 实例之间切换。
- **远程路由**：局域网、Tailscale 和 HTTPS 地址自动切换。
- **手机控制**：sideload 版本配合 Relay，可进行点击、输入、滑动、截图、剪贴板和受控批量操作。
- **通知助手**：选择性读取、总结和转发 Android 通知。
- **安全**：二维码配对、Keystore、证书固定、按通道授权和操作日志。

## Hermes-Relay CLI

实验性 CLI 让 Hermes 在已配对电脑上使用文件、终端、搜索、截图和剪贴板工具。工具执行受同意流程、补丁差异确认和 `--no-tools` 开关保护。

更完整、更新更快的功能说明见[英文功能文档](/features/)。
